hpgnum

hpgnum is the number of hugepages to be created on compute nodes. Only positive integer values are allowed for this parameter. Moreover, it has one-to-one correspondence with hpgsize. For hpgsize of 2MB, hpgnum allows 49152, and for 1GB hpgsize, 96 is the only valid value.
